[ 명령어 스크립트 정리 ]

1. 모듈 설치 명령어

(1) 익스프레스 설치
npm i express

(2) 멀터 설치
npm i multer -> 기본 최신 안정버전이 설치 됨(현재버전 1.4.5버전)
-> 이 버전에서 한글이 깨짐. ㅠㅠ
->> 안정버전 이 전에 한글이 안깨지는 버전은 1.4.4 버전임!!
npm uni multer
npm i multer@1.4.4

2. 서버 구동하기 : node js에 작성 된 server.js파일 호출
node server

->>> 해당 서버주소로 사이트 연결 가능

3. 서버 멈추기 : Ctrl+C 바로 서버 멈춤

->>> 사이트 연결 안됨
